## Install
This project requires Python and the following Python libraries installed:
* Pandas
* Numpy
* Seaborn
* Matplotlib
* Nltk
* Sklearn
* Re
* Itertools
* Mxnet

Model specific installations:
* m-BERT:
   * Bert-embedding
   * Mxnet-cu100
   * Sentence-transformers
   * Bert_embedding
   * Sentence_transformers
* XGBoost
   * Xgboost
* Stacking
   * Mlxtend
* Fastai with RoBERTa
   * Fastai_roberta
   * fastai==1.0.60
   * transformers==2.3.0
   * Torch
* Hinglish BERT
   * Torch
   * Transformers
